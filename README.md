###  Projects

####  AdLens
> A multimodal ad creative scoring engine that predicts ad engagement by jointly reasoning over ad images(CNN) and copy(transformer)
- Built a multimodal late fusion model combining ResNet-50 image branch and DistilBERT text branch, concatenated with ReLU activations and dropout 
- Added Grad-CAM heatmap overlays on CNN branch and LIME word-level highlights on the text branch to make model decisions explainable to non-technical stakeholders
- Deployed a Gradio interface to Hugging Face Spaces: upload an ad image + paste copy -> get an engagement score, visual heatmap, highlighted copy, and 3 headline improvements in real time
---
####  CruxTube 
> A full-stack AI web application that summarizes YouTube videos and playlists and enables contextual chat using transcript-based understanding.
- Built an end-to-end system using React and Node.js/Express that extracts transcripts (yt-dlp/Whisper fallback) and generates concise summaries using LLMs
- Implemented context-aware chat by retrieving relevant transcript chunks, enabling users to query videos efficiently
- Added multilingual support with optional translation to original language, and language-aware chat responses
